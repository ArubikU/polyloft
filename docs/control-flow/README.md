# Control Flow

Polyloft provides standard control flow structures for directing program execution.

- [**Conditionals**](conditionals.md) - `if`, `else`, `else if`
- [**Loops**](loops.md) - `for`, `loop`, `do-loop`
- [**Switch**](switch.md) - Pattern matching switch statements
- [**Exceptions**](exceptions.md) - `try`, `catch`, `finally` blocks
- [**Inline Syntax**](inline-syntax.md) - Single-line control flow expressions
