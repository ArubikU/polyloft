# Quick Start Guide

To get started with Polyloft quickly, check out the [Example Project](../example_project/README.md) which contains a complete starter template.

## Installation

See the [Installation Guide](../README.md#installation) for instructions on how to install Polyloft.

## Running Your First Program

1. Create a file named `hello.pf`
2. Add the following code:
   ```pf
   println("Hello, World!")
   ```
3. Run it with:
   ```bash
   polyloft hello.pf
   ```
