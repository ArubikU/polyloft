# Variables

Polyloft supports several ways to declare variables depending on mutability and scope.

- [**var**](var.md) - Mutable variables whose type can change (dynamic typing).
- [**let**](let.md) - Type-inferred variables (static typing).
- [**const**](const.md) - Compile-time constants.
- [**final**](final.md) - Runtime constants (assigned once).
